HTTP Enumeration

While Enumerating the site found an endpoint pointing to some type of pcap file which can be interesting for us. but when visiting to the actual endpoint found that the url used for accessing the pcap file is http://cap.htb/data/14 which doesnâ€™t have any packets or data. so changed the url from http://cap.htb/data/14 to http://cap.htb/data/0 which got us a pcap file which actually contains some information. So downloaded it and start analyzing it.
