HTTP Server is just running a simple single page application. After enumerating the HTTP Server for a while, running gobuster, nikto, checking for other services, nothing worked. After that looking for Services for vulnerability finds out that the Apache2 2.4.49 is vulnerable to LFI & RCE vulnerabilties with CVE-2021-41773.

After studing, what was the flaw a written a custom exploit for this specific CVE which will give us RCE on the server, you can find that exploit here.
